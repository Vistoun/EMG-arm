# EMG-arm
3D vytisknuta roboticka ruka v realné velikosti, poháněna servy. Ovladana pomocí EMG senzorů. 

## Cíl
  **Zkonstuovaní robotické ruky ovládané pomocí EMG (svalových) senzorů.**

## Hardware 
- InMoov robotic arm 
- MyoWare Muscle Sensor
- Arduino NANO
- I2C OLED display 128x64
- Rotační enkodér
- 6xMG995R Servo
